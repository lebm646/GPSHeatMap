# GPS HeatMap app by .NET MAUI

This is a cross-platform mobile application built using .NET MAUI that visualizes GPS data on a heatmap. The app allows users to load GPS data files and displays the data points on an interactive map, highlighting areas with higher concentrations of GPS points.

## Features

- Track real-time GPS location.
- Visualize GPS data on a heatmap.
